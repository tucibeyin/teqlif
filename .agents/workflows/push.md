description: commit and push all changes to GitHub with an appropriate commit message
---

1. Run `git status` to see what's changed
// turbo
2. Run `git add -A` to stage all changes
// turbo
3. Based on the changed files, craft a descriptive commit message following conventional commits format:
   - `feat:` for new features
   - `fix:` for bug fixes  
   - `refactor:` for refactoring
   - `chore:` for maintenance
   Write the message in Turkish matching the project language.
4. Run `git commit -m "<your message>"` with the crafted message
5. Run `git push` to push to GitHub
// turbo
